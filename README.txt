資展國際個人專題-youtube頻道推薦系統 : 

設計目的 : 讓使用者擺脫youtube固定推薦的內容，發覺更多有趣的頻道。

核心概念 : 設計一個隨機關鍵字系統，此系統會隨機組合同主題的單詞，選定組合後向youtube data api v3抓取符合關鍵字的頻道，取得那些頻道的基本數據後再丟入演算法進行評分，依照評分高低進行推薦。

如何啟動:
1.使用vscode開啟專案
2.下載live server
3.使用live server開啟index.html檔案即可執行

說明:
1.在同一類別(如:遊戲+中文)持續按推薦鈕，評分會越按越低，模擬真實推薦情況
2.當暫存中的頻道不夠推薦時，會自動用API再抓取一次頻道，因此可能會有評分越按越低，突然又推薦高分頻道的現象產生
